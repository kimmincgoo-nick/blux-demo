# Blux 인터랙티브 데모

자율주행 CRM 마케팅 플랫폼 **Blux**의 사용자 흐름을 직접 클릭해 볼 수 있는 인터랙티브 데모입니다.
백엔드 없이 단일 HTML로 동작하며, 팀 내부 공유 및 시연용으로 사용합니다.

## 접속 링크

**https://kimmincgoo-nick.github.io/blux-demo/**

> main 브랜치에 푸시하면 GitHub Pages가 자동 재배포합니다 (보통 1분 이내 반영).

## 데모 목적

- 영업·CS·기획 팀이 Blux의 핵심 가치(에이전트 기반 자율주행, 초개인화, 시나리오 자동화)를 **클릭하며 체감**할 수 있도록 합니다.
- 고객 미팅, 온보딩, 내부 교육 자료로 활용합니다.
- 실제 백엔드 없이 프런트엔드만으로 동작하므로 누구나 링크만으로 바로 열어볼 수 있습니다.

## 가이드 투어 5종

상단 **Guided Tours** 바 또는 첫 진입 시 환영 화면에서 선택할 수 있습니다.

| # | 투어 | 핵심 메시지 |
| --- | --- | --- |
| ① | **AI 에이전트 · 원클릭 캠페인** | 오디언스부터 발송까지 한 문장으로 |
| ② | **AI 에이전트 · 성과 리포트** | 분석 요청 한 줄로 인사이트 자동 생성 |
| ③ | **직접 세팅 (Console)** | 세밀한 컨트롤이 필요할 때 |
| ④ | **AI 오디언스 × 상품 추천** | 초개인화 마케팅의 실체 |
| ⑤ | **시나리오 · 자동화 캠페인** | 유저 여정 기반 자동 발송 흐름 |

각 투어는 코치마크(말풍선) 가이드를 따라 다음/이전 버튼으로 진행합니다. `ESC` 키로 언제든 종료할 수 있습니다.

## 사용 팁

- 데스크톱 Chrome 최신 버전에서 가장 잘 보입니다.
- 폰트는 [Pretendard](https://cactus.tistory.com/306)를 CDN으로 불러옵니다. 인터넷 연결이 필요합니다.
- 데이터는 모두 더미이며, 어떤 동작도 실제 발송·저장되지 않습니다.
